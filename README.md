
# Java Sum CLI 

This is a simple Java command-line application that takes numerical arguments and prints their total sum.  
The program does NOT run without arguments and will show an error message if the input is invalid.

## How to Use

### 1. Compile the program:
javac Sum.java

### 2. Run the program with numbers:
java Sum 5 10 3

### Example Output:
Sum = 18
